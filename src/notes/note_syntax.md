## Basic Syntax

These are the elements outlined in John Gruberâ€™s original design document. All Markdown applications support these elements.

### Heading

# H1
## H2
### H3
#### H4

### Bold

**bold text**

### Italic

*italicized text*

### Blockquote

> blockquote is een quote
>
>> die ook genest kan
>
> mooi toch?

### Ordered List

1. First item
2. Second item
3. Third item

### Unordered List

- First item
- Second item
- Third item

### Code

`code`

### Footnote

Here's a sentence with a footnote. [^1]

[^1]: This is the footnote.

### Horizontal Rule

---

### Link

[Markdown Guide](https://www.markdownguide.org)

### Image
